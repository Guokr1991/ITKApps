  1. Introduction => back to top

    2. This tool has been created in order to rigidly register different modalities images. The method is made of two steps:

    • pre-registration step, which is a landmark-based registration technique. That method minimize the square distances between the landmarks.
    • registration step, which uses the mutual information registration technique implemented in the Insight Toolkit (www.itk.org)

  2. How to use the software => back to top

    3. Is that the first time you use this tool ??? well, here is the steps to follow in order to register images.

    1) Select a reference image by clicking on the "Select Reference image..." button (upper left corner of the interface). Once you click on that button, a file browser window will appear. Select the file you want to use as a fixed image, and click "OK" when you are ready to go on.

    2) Select an image to register with the reference image you have previously selected by clicking on the "Select Moving image..." button (left side of the interface). Proceed like the previous step with the file browser.

    At that point, you have two different options. If you want to run the registration only, without previous pre-registration, follow the following step.

    3) If you want to register the images, without running the pre-registration, just click now on the register button, located in the lower part of the interface.

    If you need to run the pre-registration on your images first, and then apply the registration on the pre-registered data, follow the instructions below:

    3) If you want to pre-register the images before running the registration method, you have to select four landmarks on both the reference, and the moving image. To do so, click on one of the button labelled 'Landmark x', then click on the image viewer on the rigth side of the button at the spot where you want to place the landmark x. Once a landmark is set up, it will be checked in the inferface. Follow the same protocol for each of the landmarks. The 'Go to...' button located on the right side of each landmark button allows you to jump to the slice where the corresponding landmarks has been placed. Once all the landmarks are checked (selected), you can click the 'Pre-Register' button which is in the lower part of the interface.

    4) Run the registration by clicking the 'Register' button right below the 'Pre-Register' button.

     

  3. How to save your work => back to top
    • Saving the pre-registered image
      To save the pre-registered image, go to the'File' menu, and select 'Save pre-registered moving image...'. A file browser window will show up, and allow you to select the directory in which you want to save the pre-registered image. You can type in the name of the file in the 'Filename' field located at the bottom of the file browser. Click 'OK' and the file will be saved.
    • Saving the registered image
      To save the registered image, go to the 'File' menu, and select 'Save registered moving image...'. Then follow the instructions already mentioned above for the pre-registered image.
    • Saving your study
      You can also save your current study, which means the images selected, the landmarks, and the pre-registration as well as the registration. In order to save the study in progress, go to the 'File' menu, click on 'Save study...', and then follow the instructions provided above.

     

  4. How to load your previous work
    5. => back to top
    • Loading a study
      Go to the 'File' menu, click on 'Load study...' and select the study you want to load up using the file browser that shows up. Click 'Ok' to confirm your choise, and wait a few seconds for the study to load into the program.
    • Loading landmarks
      To load landmarks for an image, click on the 'Load landmarks...' button located on the left side of the window where the image is displayed. Select the file you want to load up using the file browser, and click 'OK' to confirm your choice. The landmarks loaded appear in the corresponding display.
