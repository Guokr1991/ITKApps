ITK-SNAP Pre-Alpha Readme

Thank you for your interest in the ITK-SNaP (Snake Automatic Partitioning) Software!

This is an early pre-alpha version. Cognitica Corp. is working to deliver a working version of this software by Sep. 30, 2003. Until then, please expect a level of instability in the software. A list of known bugs can be viewed at

  • http://www.cognitica.com/download/snap/bugs.html

You will need a 3D image to use this software. Images with which SNAP is known to work can be downloaded from

  • http://www.cognitica.com/download/snap/images.html

Please direct all questions to Paul Yushkevich (pauly@cognitica.com)
