Tumor Segmentation Evaluation
-----------------------------

The directory contains the Volview Plugins for Tumor Mesurement and Comparison

INSTALL NOTES

1.Donwload CMake from www.cmake.org and ITK(Insight ToolKit) from www.itk.org

2.Install ITK(see www.itk.org for installation notes)

3.Run CMake and select the directory where the plugins are located and the directory where ITK binaries is located

4.Run your c++ compiler with the project generated by CMake and compile the project.

5.If is not specified in the CMake configuration where should be the plugins compiled then manually copy the compiled plugins to Volview Plugins directory

6.Open Volview application and locate them in Filters tab or select from menu View/Filters

That's it
